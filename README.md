## Built with

- vanilla js

---

## Features

- dark / light mode toggle (saves to localStorage)
- typing animation in the hero
- sections fade in as you scroll
- active nav link highlights based on scroll position
- hamburger menu on mobile
- back to top button
- form validation without a single alert()
- 🐯 favicon using a one-liner svg trick
- console easter egg for people who inspect the page

---

## Run it locally

Just clone it and open the file,

```bash
git clone https://github.com/ANMOLGUPTA-24/Tiger_portfolio
cd Tiger_portfolio
```

then open `index.html` in your browser. or use live server in vs code if you want auto-reload.

---

## To add your own image

drop a file called `tiger.jpg` in the same folder. it'll automatically show up in the about section. if there's no file it falls back to the 🐯 emoji so nothing breaks.

---

## To add your resume

drop a `resume.pdf` in the same folder. the download button in the hero links to it.

---

## Structure

Everything's in `index.html` and `style.css`. built this in sections like hero, about, projects, skills, testimonials, contact. Each section took roughly a day to build and style properly.

## Why tiger

W was going to make a normal portfolio but that seemed boring. made it for a tiger instead. The projects are fictional, the testimonials are fictional, the "ROAR-ROAR-ROAR" phone number is fictional.

When i actually deploy my real portfolio i'll swap in my own content and keep the same design.
